# netinfo.sh 脚本工具

## 使用说明
1. **普通运行**（仅显示已激活）：
   ```Bash
    netinfo
    ```
   
2. **显示全部**（包含未激活）：
   ```Bash
    netinfo -a
   ```

## 设计细节说明
* **nmcli device status**: 改用设备状态作为主数据源，相比 con show 更加底层，能更准确地识别即使没建立连接的物理网卡（DEVICE）。

* **动态提示**: 通过 get_interface_info 函数复用逻辑。如果系统没有任何激活的连接（例如网线被拔掉），它会准确显示“未找到激活的网络接口”，而不是输出一个空表头。

* **参数扩展**: 如果你之后想增加对 IPv6 的支持，只需在 nmcli 的提取部分增加对应的 IP6.ADDRESS 即可。
